# 📋 API 서비스 활용 가이드

본 문서는 웨더리프 서비스에서 사용하는 각 API 엔드포인트의 역할과 호출 시점을 정리한 문서입니다.

| 번호   | 엔드포인트명          | 연동 데이터          | 사용처 및 시점                                                                           |
| :----- | :-------------------- | :------------------- | :--------------------------------------------------------------------------------------- |
| **1**  | `/api/weather`        | **초단기실황**       | 앱 실행 시 초기화. **현재 기온/날씨** 표시의 핵심 데이터입니다.                          |
| **2**  | `/api/ultra-srt-fcst` | **초단기예보**       | 실황 데이터 지연 시(매시 정각~40분) **현재 날씨를 실시간으로 보정**하기 위해 사용합니다. |
| **3**  | `/api/forecast`       | **단기예보**         | 오늘부터 모레까지의 **시간대별 날씨와 최저/최고 기온**을 구성합니다.                     |
| **4**  | `/api/mid-land`       | **중기육상예보**     | **주간 예보** 영역의 3일~10일 뒤 날씨 아이콘 및 상태 메시지에 사용됩니다.                |
| **5**  | `/api/mid-ta`         | **중기기온조회**     | 주간 예보 영역의 3일~10일 뒤 **예상 온도 정보**를 제공합니다.                            |
| **6**  | `/api/tm-coord`       | **TM 좌표 변환**     | 사용자의 위경도를 미세먼지 측정소 조회용 **TM 좌표**로 변환하는 중간 단계입니다.         |
| **7**  | `/api/nearby-station` | **근접 측정소 목록** | 사용자와 가장 가까운 **대기오염 측정소의 이름**을 찾아내는 데 사용합니다.                |
| **8**  | `/api/sido-dust`      | **시도별 측정정보**  | 측정소 데이터 장애 시 해당 **지차체(시/도) 전체 평균**을 보여주는 백업 로직입니다.       |
| **9**  | `/api/dust`           | **실시간 미세먼지**  | 측정소를 통해 가져온 **미세먼지/초미세먼지 농도와 등급**을 표시합니다.                   |
| **10** | `/api/uv`             | **자외선 지수**      | 실시간 자외선 지수를 확인하여 **선크림, 양산 등 준비물 추천**을 결정합니다.              |
| **11** | `/api/air-diffusion`  | **대기정체 지수**    | 환기나 야외 활동에 참고할 수 있는 **대기 정체 정도**를 상세 카드에 표시합니다.           |
| **12** | `/api/kakao-address`  | **카카오 주소 변환** | GPS 숫자 좌표를 화면 상단에 표시되는 **한글 지역 명칭**으로 변환합니다.                  |

## 🔄 데이터 흐름 예시

### [날씨 정보 조회]

1. 유저 위치 GPS 획득
2. `/api/weather` + `/api/ultra-srt-fcst` 호출 → **현재 날씨 완성**
3. `/api/forecast` 호출 → **오늘의 시간별 예보 완성**
4. `/api/mid-land` + `/api/mid-ta` 호출 → **주간 예보 완성**

### [미세먼지 정보 조회]

1. `/api/tm-coord` 호출 (좌표 변환)
2. `/api/nearby-station` 호출 (가까운 측정소 이름 획득)
3. `/api/dust` 호출 → **최종 미세먼지 등급 표시**

### [지능형 추천]

1. `/api/uv` 호출 (현재 시각 기준)
2. `getRecommendedItems` 로직 실행 → **선크림, 우산 등 아이콘 노출**
